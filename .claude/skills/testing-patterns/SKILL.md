  Aktiviere wenn: Tests geschrieben werden sollen, TDD genutzt wird,
  jemand fragt wie man etwas testet oder Mocks erstellt.

- AAA-Pattern: Arrange, Act, Assert
- Test zuerst schreiben (TDD: Red → Green → Refactor)
- Externe Dependencies mocken, keine internen
- Test-Namen: beschreiben was getestet wird und was erwartet wird
- Integration Tests vs Unit Tests — wann was
