  Aktiviere wenn: Code reviewt werden soll, ein PR fertig ist,
  jemand fragt "ist das gut so" oder "kannst du das prüfen".
---

**Checkliste für Code-Reviews:**
- Lesbarkeit und Verständlichkeit
- Fehlerbehandlung vollständig?
- Tests vorhanden?
- Sicherheitsprobleme (hardcoded secrets, SQL injection, XSS)
- Performance-Probleme
- Kommentare auf Deutsch und sinnvoll?
